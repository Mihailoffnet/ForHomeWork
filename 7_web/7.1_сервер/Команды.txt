подключиться к серверу по ip и паролю:
ssh root@80.78.248.233

создать конфиг:
touch ~/.ssh/config

содержимое конфига:
Host testserver
    HostName 80.78.248.233
    Port 22
    User root

скопировать SSH на сервер для доступа без пароля
ssh-copy-id root@80.78.248.233

подлкючиться без пароля по псевдониму 
 ssh testserver
